# html-assignment-2# HTML II Assignment - Multi-Page Semantic Website

## Project Description
This is a two-page connected website built to demonstrate a solid understanding of HTML5 semantic elements, forms, media tags, interactive elements, and navigation links.

## Features Implemented
- **Semantic Structure:** Organized using `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, and `<footer>`.
- **Media & Interactive Content:** Embedded audio, video, and a collapsible section using `<details>`.
- **External Embeds:** Integrated a Google Map and a YouTube video using `<iframe>` tags.
- **Form Elements:** Created a complete registration form with text inputs, radio buttons, checkboxes, a dropdown selection, and a textarea.
